# MJ Web Boilerplate
Boilerplate for web projects using gulp, jQuery and Bootstrap 4.

## How to

1. Clone this repository.
2. ``` cd ``` your way to the folder.
3. ``` npm install ```
4. ``` npm run dev ```

You have a webserver running at ``` localhost:8080 ```
